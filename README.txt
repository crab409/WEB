현제 서버 파일 
    server.js



서버 실행 터미널 명령어 
    node server.js

    혹은
    nodemon server.js

서버 중지 터미널 명령어 
    키보드 단축 : ctrl + C


설치된 라이브러리 
    "bcrypt": "^5.1.1",
    "connect-mongo": "^5.1.0",
    "ejs": "^3.1.10",
    "express": "^4.21.1",
    "express-session": "^1.18.1",
    "method-override": "^3.0.0",
    "mongodb": "^5.9.2",
    "passport": "^0.7.0",
    "passport-local": "^1.0.0"

    +자세한건 package.json의 dependencies 항목 참조



각 파일 위치
    /server.js : 
        서버 파일 

    /node_modules/~/ : 
        설치된 라이브러리가 저장되는 폴더

    /package.json : 
        전체 프로그램 상태 요약

    /package_lock.json : 
        더 자세한 전체 프로그램 상태 요약

    /views/~.ejs : 
        HTML + JavaScript 파일 

    /public/css/~.css : 
        CSS파일 

    /public/javaScript/~.js : 
        JavaScript 파일
